Whippy

A fluid grid system

$gutter: 20px; $max-columns: 8; @import ‘whippy’; .grid-container { @include whip; } .grid-column { @include col($cols, [$context]); }

Whippy is a handful of SASS/LESS/Stylus mixins for creating simple, fluid grid columns.

How it works

The width of a grid cell is its number of columns divided by its container’s columns. Gutters are added with padding, and a container pulls left and right gutters back out with negative margins. This way content inside whippy columns will always align with non-grid content outside.

Keep in mind

  • Grid containers must be width: auto for the negative margin trick to work
  • Nest elements inside grid columns to add backgrounds and borders
